### How do I get set up? ###

* Для установки:
	* composer install
	* Создаем .env файл из образца
	* php artisan key:gen
	* npm i
	* cd public/
	* bower install (for AdminLTE template)
	* php artisan migrate
	* php artisan db:seed
* Done...

### Vuejs commands (webpack) ###

* Для билда проекта
	* npm run dev (development mode)
	* npm run watch (watcher)
	* npm run production (production mode)
	* npm run hot (hot reload)
